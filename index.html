Console v1.0
➜ /tracker @usuario // Añadir objetivo a tu lista privada
➜ /tracker_off // Pausar notificaciones temporalmente
➜ /tracker_on // Reanudar notificaciones pausadas
➜ /tracker_list // Ver tu lista de usuarios vigilados
➜ /tracker_remove @usuario // Dejar de vigilar a un usuario
➜ /tracker_clear // Borrar todos tus datos de rastreo
➜ /tracker_help // Mostrar guía de uso rápido

Privacidad por diseño

Ni los administradores del servidor ni los dueños del bot cotilleamos tus listas. Tus datos de rastreo son privados y se gestionan bajo una base de datos segura.
